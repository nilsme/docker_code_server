# README

Build and deploy `code-server` for local use with Fedora 34 as OS.

## Build images

```Shell script
docker-compose build
```

## Start code-server

```Shell script
docker-compose up -d
```

## Access code-server

Browse to <https://127.0.0.1:8080> or <https://localhost:8080>

## Stop code-server

```Shell script
docker-compose down
```
